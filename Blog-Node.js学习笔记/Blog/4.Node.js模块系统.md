# 模块系统

> node是如何做到一次执行多个文件的呢？这里就涉及到多文件了

1. require是一个方法，它的作用是用来加载模块的。
2. 在Node中，模块分为三种。
 * 具名的核心模块，如：fs、http等。
 * 用户自己编写的文件模块（JS文件）。
 *
 
请看如下例子:
```js
console.log('a start');
require('./b.js');
console.log('a end');
//执行顺序为 a start，b文件执行，a end。
```

下面的例子: 执行`a.js`文件输出什么呢？
```js
//a.js文件
console.log('a start');
require('./b.js');
console.log('a end');

//b.js文件
console.log('b start');
require('./c.js');
console.log('b end');

//c.js文件
console.log('c.js文件执行了');
```
